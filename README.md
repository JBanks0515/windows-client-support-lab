# Windows Client Support Lab

## Overview
This lab demonstrates hands-on Windows client support scenarios, including:

- Cloning and configuring a Windows client VM
- Managing local and domain user accounts
- Troubleshooting login issues
- Diagnosing network connectivity problems

The lab environment consists of:
- **Domain Controller (DC):** Windows Server with Active Directory
- **Client VM:** Windows 10/11 joined to the domain
- **Snapshots:** Clean baselines for testing

## Objectives
- Build practical experience supporting Windows end-user systems
- Compare local vs domain user accounts
- Simulate and resolve login failures
- Identify and fix network connectivity issues

## Prerequisites
- VirtualBox or VMware installed
- Domain Controller VM running Windows Server
- Client VM running Windows 10/11 joined to the domain
- Admin access on both client and DC
